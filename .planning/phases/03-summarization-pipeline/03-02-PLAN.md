**1. Data class for pipeline result:**

```python
@dataclass
class BriefingResult:
    article_count: int
    file_path: str | None      # None if no articles
    briefing_id: int | None    # None if no articles
    summary_text: str           # Formatted briefing content or "no articles" message
```

**2. Main function — `async def run_briefing_pipeline() -> BriefingResult`:**

Follow SOP Section 6-1 process exactly:

**Step 1: Fetch unbriefed articles from DB**
```sql
SELECT id, title, url, source_name FROM articles
WHERE is_briefed = 0
ORDER BY collected_at ASC
```
Use `db = await get_db()` from `src.storage.database`.

**Step 2: Handle zero articles case (SOP Section 7-2)**
If no articles found:
- Log: `logger.info("No unbriefed articles found")`
- Return `BriefingResult(article_count=0, file_path=None, briefing_id=None, summary_text="새로운 기사가 없습니다.")`
- Do NOT create .md file (per SOP Section 7-2)
- Do NOT insert into briefings table (per SOP Section 7-2)

**Step 3: Call summarizer**
- Convert DB rows to `ArticleForSummary` objects
- Call `summary_lines = await summarize_articles(article_list)`
- This returns either Claude summaries or fallback title list (handled by Plan 01)

**Step 4: Format .md content (SOP Section 6-3)**
Build the full markdown content per SOP format. The template must include:
- Header line: `# 뉴스 브리핑 | {YYYY-MM-DD HH:MM}`
- Metadata line: `> 대상 기사: {N}건`
- A horizontal rule separator (three dashes)
- The summary lines from Claude API
- Another horizontal rule separator (three dashes)
- Footer line: `총 {N}건 | 생성: {YYYY-MM-DD HH:MM} KST`

Use `datetime.now(ZoneInfo("Asia/Seoul"))` for KST timestamps. Import `from zoneinfo import ZoneInfo`.

Build the content string using an f-string or string concatenation with `\n` joins. The horizontal rule separators should be `"---"` strings embedded in the template.

**Step 5: Save .md file (SOP Section 6-4, STR-01)**
- Get briefing folder from settings: `folder = await get_setting("briefing_folder") or "briefings"`
- Create directory: `Path(folder).mkdir(parents=True, exist_ok=True)`
- Filename: `now.strftime("%Y-%m-%d_%H-%M") + ".md"` where `now` is the KST datetime
- Write content with UTF-8 encoding
- Log: `logger.info(f"Briefing saved to {filepath}")`

**Step 6: Mark articles as briefed (SUM-04)**
- Insert briefing record first to get the briefing_id:
```sql
INSERT INTO briefings (generated_at, article_count, content_md, file_path, delivered)
VALUES (?, ?, ?, ?, 0)
```
- `generated_at` = KST datetime ISO format
- `article_count` = number of articles
- `content_md` = the full markdown text
- `file_path` = relative path to .md file
- Get the `lastrowid` as `briefing_id`

- Then update all articles:
```sql
UPDATE articles SET is_briefed = 1, briefing_id = ? WHERE id IN ({placeholders})
```
- Use the list of article IDs from Step 1
- Commit the transaction: `await db.commit()`
- Log: `logger.info(f"Marked {len(article_ids)} articles as briefed (briefing_id={briefing_id})")`

**Step 7: Return result**
```python
return BriefingResult(
    article_count=len(article_list),
    file_path=filepath,
    briefing_id=briefing_id,
    summary_text=md_content,
)
```
