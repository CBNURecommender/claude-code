Create `src/summarizer/briefing.py` with the following implementation. Use `anthropic.AsyncAnthropic` client (per stack decision — async-native, Anthropic SDK >=0.40.0).

**1. Data class for input:**

```python
@dataclass
class ArticleForSummary:
    title: str
    url: str
    source_name: str
```

**2. Module constants:**

```python
MODEL = "claude-sonnet-4-20250514"
MAX_RETRIES = 3
RETRY_DELAY_SECONDS = 30
```

**3. System prompt (from SOP Section 6-2, verbatim):**

```
당신은 IT/반도체 뉴스 요약 전문가입니다.
각 기사를 아래 형식으로 한줄 요약하세요:
[핵심키워드] 한줄 요약 — 출처
핵심키워드는 기사의 가장 중요한 주제어 1~2개를 대괄호 안에 넣습니다.
한줄 요약은 20~40자 이내로 핵심만 담습니다.
출처는 뉴스 소스 이름입니다.
```

**4. User prompt builder — `_build_user_prompt(articles: list[ArticleForSummary]) -> str`:**

Format articles as numbered list per SOP Section 6-2:
```
아래 기사들을 각각 한줄 요약해주세요.

1. [전자신문] 삼성전자, HBM4 양산 본격화...
   URL: https://...
2. [TrendForce] TSMC Reports 2nm Yield Breakthrough
   URL: https://...
```

Each line: `{i}. [{source_name}] {title}\n   URL: {url}`

**5. Core function — `async def summarize_articles(articles: list[ArticleForSummary]) -> str`:**

- Import config via `from src.utils.config import get_config` to get ANTHROPIC_API_KEY
- Create `anthropic.AsyncAnthropic(api_key=config.ANTHROPIC_API_KEY)` — create client per call (not module-level) so config is loaded at runtime
- Build user prompt from articles using `_build_user_prompt()`
- Call `client.messages.create()` with:
  - `model=MODEL`
  - `max_tokens=4096`
  - `system=SYSTEM_PROMPT`
  - `messages=[{"role": "user", "content": user_prompt}]`
- Extract response text: `response.content[0].text`
- Strip whitespace and return the summary text
- **Retry logic:** Wrap the API call in a retry loop (MAX_RETRIES=3). On `anthropic.APIError` or `anthropic.APIConnectionError`, log warning with retry count, `await asyncio.sleep(RETRY_DELAY_SECONDS)`, and retry. After all retries exhausted, call `_build_fallback(articles)` and return that instead.
- Log: `logger.info(f"Summarized {len(articles)} articles via Claude API")` on success
- Log: `logger.error(f"Claude API failed after {MAX_RETRIES} retries, using fallback")` on fallback

**6. Fallback function — `def _build_fallback(articles: list[ArticleForSummary]) -> str`:**

Returns raw article titles as a simple list (SUM-05: fallback to raw title list):
```
[요약 실패 — 원본 제목 목록]

- 삼성전자, HBM4 양산 본격화... — 전자신문
- TSMC Reports 2nm Yield Breakthrough — TrendForce
```

Format: `- {title} — {source_name}` for each article.

**7. Update `src/summarizer/__init__.py`:**

Add: `from src.summarizer.briefing import summarize_articles, ArticleForSummary`

**Important notes:**
- Do NOT use `requests` or sync HTTP — use `anthropic.AsyncAnthropic` which uses httpx internally
- Do NOT create one API call per article — all articles go in a single batched call (SUM-03)
- System prompt is in Korean to ensure Korean output regardless of article language (SUM-02)
- Use `get_logger("summarizer.briefing")` for logging
