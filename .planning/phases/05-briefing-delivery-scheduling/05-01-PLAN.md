**Function 1: `split_message(text: str, max_length: int = MAX_MESSAGE_LENGTH) -> list[str]`**
Split a long message into chunks that respect Telegram's 4096-char limit. Split at line boundaries (newline characters) to avoid breaking mid-sentence. Algorithm:
1. If `len(text) <= max_length`, return `[text]`
2. Split text into lines
3. Accumulate lines into chunks. When adding the next line would exceed max_length, start a new chunk
4. If a single line exceeds max_length (unlikely but defensive), hard-split it at max_length
5. Return list of chunks (each <= max_length)

**Function 2: `async def send_to_all_users(bot: telegram.Bot, text: str) -> int`**
Send a message (potentially split) to all registered chat_ids. Returns the count of successfully sent chat_ids.
1. Read `telegram_chat_ids` from settings via `get_setting('telegram_chat_ids')`
2. Parse as JSON array: `chat_ids = json.loads(raw)` — if None or empty JSON array `[]`, log warning "No registered chat_ids, skipping delivery" and return 0
3. Split the text using `split_message(text)`
4. For each `chat_id` in `chat_ids`:
   - For each chunk in split messages:
     - `await bot.send_message(chat_id=int(chat_id), text=chunk)`
   - Wrap in try/except to catch `telegram.error.TelegramError` — log warning with chat_id and error, continue to next user
5. Return count of successfully delivered users

**Function 3: `async def deliver_briefing(bot: telegram.Bot, briefing_content: str, article_count: int) -> int`**
High-level delivery function called by the scheduler or /briefing command. Returns count of users delivered to.
1. If `article_count == 0`:
   - Send "no new articles" message to all users: `"📭 새로운 기사가 없습니다."` (per SOP Section 7-2)
   - Do NOT create .md file, do NOT record in briefings table (caller handles this logic based on article_count)
   - Return `await send_to_all_users(bot, no_articles_msg)`
2. If `article_count > 0`:
   - The `briefing_content` parameter already contains the formatted Telegram message (header + summaries + footer, formatted per SOP Section 7-1)
   - Return `await send_to_all_users(bot, briefing_content)`
3. Log: `logger.info(f"Briefing delivered to {count} users ({article_count} articles)")`

**Function 4: `def format_briefing_message(summaries: str, article_count: int, timestamp_str: str, md_filename: str | None = None) -> str`**
Format the briefing content into the Telegram message format per SOP Section 7-1:
```
📰 뉴스 브리핑 | {timestamp_str}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

{summaries}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
총 {article_count}건{' | 저장: ' + md_filename if md_filename else ''}
```
