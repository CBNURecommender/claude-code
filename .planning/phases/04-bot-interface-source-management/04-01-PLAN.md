Create `src/storage/queries.py` with the following async functions. All functions use `await get_db()` from `src.storage.database` to get the shared connection. Import `get_logger` from `src.utils.logger` with logger name `"storage.queries"`.

**Source CRUD:**

1. `async def get_source_by_name(name: str) -> aiosqlite.Row | None`:
   - `SELECT * FROM sources WHERE name = ? COLLATE NOCASE`
   - Returns the row or None

2. `async def get_source_by_id(source_id: int) -> aiosqlite.Row | None`:
   - `SELECT * FROM sources WHERE id = ?`
   - Returns the row or None

3. `async def add_source(name: str, url: str) -> int`:
   - `INSERT INTO sources (name, url) VALUES (?, ?)`
   - Commit, return `cursor.lastrowid`
   - Let sqlite UNIQUE constraint on url raise `aiosqlite.IntegrityError` if duplicate URL

4. `async def remove_source(source_id: int) -> None`:
   - `DELETE FROM sources WHERE id = ?`
   - Commit (CASCADE deletes keywords automatically)

5. `async def list_sources() -> list[aiosqlite.Row]`:
   - `SELECT s.*, (SELECT COUNT(*) FROM source_keywords sk WHERE sk.source_id = s.id) AS keyword_count FROM sources s ORDER BY s.name`
   - Returns list of rows with keyword_count appended

6. `async def enable_source(source_id: int) -> None`:
   - `UPDATE sources SET enabled = 1 WHERE id = ?`
   - Commit

7. `async def disable_source(source_id: int) -> None`:
   - `UPDATE sources SET enabled = 0 WHERE id = ?`
   - Commit

**Per-source keyword CRUD:**

8. `async def add_source_keyword(source_id: int, keyword: str) -> bool`:
   - `INSERT OR IGNORE INTO source_keywords (source_id, keyword) VALUES (?, ?)`
   - Commit, return `cursor.rowcount > 0` (False if duplicate)

9. `async def remove_source_keyword(source_id: int, keyword: str) -> bool`:
   - `DELETE FROM source_keywords WHERE source_id = ? AND keyword = ? COLLATE NOCASE`
   - Commit, return `cursor.rowcount > 0` (False if not found)

10. `async def list_source_keywords(source_id: int) -> list[str]`:
    - `SELECT keyword FROM source_keywords WHERE source_id = ? ORDER BY keyword`
    - Returns list of keyword strings

11. `async def clear_source_keywords(source_id: int) -> int`:
    - `DELETE FROM source_keywords WHERE source_id = ?`
    - Commit, return `cursor.rowcount` (number deleted)

**Global keyword CRUD:**

12. `async def add_global_keyword(keyword: str) -> bool`:
    - `INSERT OR IGNORE INTO global_keywords (keyword) VALUES (?)`
    - Commit, return `cursor.rowcount > 0`

13. `async def remove_global_keyword(keyword: str) -> bool`:
    - `DELETE FROM global_keywords WHERE keyword = ? COLLATE NOCASE`
    - Commit, return `cursor.rowcount > 0`

14. `async def list_global_keywords() -> list[str]`:
    - `SELECT keyword FROM global_keywords ORDER BY keyword`
    - Returns list of keyword strings

**Chat ID management:**

15. `async def register_chat_id(chat_id: int) -> bool`:
    - Uses `get_setting("telegram_chat_ids")` to load JSON array
    - If `str(chat_id)` not in the list, append it and `set_setting("telegram_chat_ids", json.dumps(updated_list))`
    - Return True if newly added, False if already registered

16. `async def get_chat_ids() -> list[int]`:
    - Uses `get_setting("telegram_chat_ids")` to load JSON array
    - Returns list of ints (parse each string to int)
    - Returns empty list if setting is missing or empty array

**Pending articles count (for /status):**

17. `async def count_pending_articles() -> int`:
    - `SELECT COUNT(*) FROM articles WHERE is_briefed = 0`
    - Returns the count

18. `async def count_sources_by_status() -> tuple[int, int]`:
    - `SELECT enabled, COUNT(*) FROM sources GROUP BY enabled`
    - Returns (active_count, disabled_count)
