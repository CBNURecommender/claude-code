**Helper function:**
```python
async def _resolve_source(update: Update, args: list[str], min_args: int, usage: str) -> aiosqlite.Row | None:
    """Validate args count and resolve source by name. Replies with error if invalid. Returns source row or None."""
```
- If `len(args) < min_args`: reply with usage string, return None
- Call `queries.get_source_by_name(args[0])`
- If not found: reply "소스를 찾을 수 없습니다: {args[0]}", return None
- Return the source row

**1. `/add_keyword <source_name> <keyword>` handler** (KWD-01):
- Use `_resolve_source(update, args, 2, "사용법: /add_keyword <소스이름> <키워드>")`
- Join args[1:] as keyword (in case keyword has spaces — but typically single word)
- Actually: keyword is `args[1]` (single word per SOP). If user wants multi-word, they should quote — but SOP shows single words. Use `" ".join(args[1:])` to be safe.
- Call `queries.add_source_keyword(source['id'], keyword)`
- If returned False (duplicate): reply "이미 등록된 키워드입니다: {keyword}"
- Get updated keyword list via `queries.list_source_keywords(source['id'])`
- Reply (SOP format):
```
'{source_name}'에 키워드 추가: {keyword}
현재 키워드: {', '.join(keywords)} ({len(keywords)}개)
```

**2. `/remove_keyword <source_name> <keyword>` handler** (KWD-02):
- Same source resolution with min_args=2
- keyword = `" ".join(args[1:])`
- Call `queries.remove_source_keyword(source['id'], keyword)`
- If returned False: reply "키워드를 찾을 수 없습니다: {keyword}"
- Get updated list
- Reply:
```
'{source_name}'에서 키워드 삭제: {keyword}
현재 키워드: {', '.join(keywords)} ({len(keywords)}개)
```
- If no keywords left, show: "현재 키워드: 없음 (전체 수집 모드)"

**3. `/list_keywords <source_name>` handler** (KWD-03):
- Resolve source with min_args=1
- Get keywords via `queries.list_source_keywords(source['id'])`
- If empty: reply "'{source_name}'의 키워드: 없음 (전체 수집 모드)"
- If has keywords: reply "'{source_name}'의 키워드 ({count}개):\n{newline-joined list with bullet points}"

**4. `/clear_keywords <source_name>` handler** (KWD-04):
- Resolve source with min_args=1
- Call `queries.clear_source_keywords(source['id'])`
- Reply: "'{source_name}'의 키워드 {deleted_count}개 삭제 완료\n전체 수집 모드로 전환되었습니다."
- If deleted_count == 0: reply "'{source_name}'에 등록된 키워드가 없습니다."

**5. `/add_global <keyword>` handler** (KWD-05):
- Check args has at least 1. Usage: "사용법: /add_global <키워드>"
- keyword = `" ".join(args)`
- Call `queries.add_global_keyword(keyword)`
- If False: reply "이미 등록된 전체 키워드입니다: {keyword}"
- Get updated list
- Reply: "전체 공통 키워드 추가: {keyword}\n현재 전체 키워드: {', '.join(keywords)} ({count}개)"

**6. `/remove_global <keyword>` handler** (KWD-05):
- Check args. Usage: "사용법: /remove_global <키워드>"
- keyword = `" ".join(args)`
- Call `queries.remove_global_keyword(keyword)`
- If False: reply "키워드를 찾을 수 없습니다: {keyword}"
- Reply: "전체 공통 키워드 삭제: {keyword}"

**7. `/list_globals` handler** (KWD-05):
- Call `queries.list_global_keywords()`
- If empty: reply "등록된 전체 공통 키워드가 없습니다."
- Reply: "전체 공통 키워드 ({count}개):\n{bullet list}"

**8. `register_keyword_handlers(app: Application) -> None`:**
Register CommandHandlers for all 7 commands: add_keyword, remove_keyword, list_keywords, clear_keywords, add_global, remove_global, list_globals.
