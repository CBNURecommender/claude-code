**1. Chat ID auto-capture middleware** (BOT-05):
```python
async def auto_register_chat_id(update: Update, context: ContextTypes.DEFAULT_TYPE) -> None:
    """MessageHandler callback that runs on every message to capture chat_id."""
```
- If `update.effective_chat` is None, return early
- Call `queries.register_chat_id(update.effective_chat.id)`
- If newly registered (returns True), log: `logger.info(f"New chat_id registered: {update.effective_chat.id}")`
- This handler must NOT consume the update (other handlers should still process it). Register it in group -1 so it runs before command handlers but does not block them.
