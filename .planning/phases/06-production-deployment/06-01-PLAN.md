**deploy/deploy.sh** (per DEP-01, DEP-04):
- Shebang: #!/bin/bash, set -e
- Variables: SERVER="jihoon@34.172.56.22", REMOTE_DIR="/home/jihoon/news-briefing", SERVICE_NAME="news-briefing"
- Step 1 - rsync with --delete and these excludes (per DEP-04):
  - --exclude '.env'
  - --exclude 'data/'
  - --exclude 'briefings/'
  - --exclude 'logs/'
  - --exclude '__pycache__/'
  - --exclude '*.pyc'
  - --exclude 'venv/'
  - --exclude '.git/'
  - --exclude '.planning/'

  - mkdir -p data briefings logs

  - sudo cp deploy/news-briefing.service /etc/systemd/system/

  - sudo systemctl enable news-briefing
  - sudo systemctl restart news-briefing
  - sleep 2, show status
- Make deploy.sh executable (chmod +x)
